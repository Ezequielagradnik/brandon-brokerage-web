# Brandon Brokerage Group — Homepage Concepts

Three premium homepage directions sharing one editorial language (Bodoni + gold hairlines), each with a distinct three.js signature.

- **A — Midnight & Gold** (`brandon-concept-a.dc.html`): navy, rotating wireframe globe.
- **B — Ivory & Sapphire** (`brandon-concept-b.dc.html`): light, animated silk-gradient shader.
- **C — Obsidian & Champagne** (`brandon-concept-c.dc.html`): near-black, faceted gold gem.

Open `Brandon Concepts.dc.html` (the index) to compare and jump into each full homepage.

## GitHub Pages
These are static HTML files. Enable Pages (Settings → Pages → deploy from branch `main`, root) and browse to the index file. Keep `support.js` and `assets/` alongside the HTML.
